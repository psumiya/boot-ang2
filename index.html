Hello to Angular 2 with Spring Boot.

This project provides an integration of the Hello World type examples available on the Angular 2 QuickStart and the Spring Boot Getting Started Pages. The inspiration is to have the enterprise Java folks have a starting point for when they are new to Angular 2, Spring Boot, or both.

Additional Documentation

A detailed explanation is available at http://www.algoclinic.com/angular-2-with-spring-boot.html
